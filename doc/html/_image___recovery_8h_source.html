155 //We need to enumerate the boxes, and pairings we are working with.
156 //Consistency is key. In X_l there are 4^l cells. In Y_l there are
157 //4^{L-l}. The number of pairs is 4^L=N^2 for any level.
158 //Each pair (A,B) at a scale corresponds to a number n=0,...N^2-1.
159 //A=0,...,4^l-1, and B=0,...,4^{L-l}-1. A simple map
160 //is to take the first 2l bits of n to be A, and the last 2(L-l)
161 //bits to be B n->(n/4^l-1,4n^l)=(A,B).
